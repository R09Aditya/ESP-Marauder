# ESP-Marauder
ESP-Marauder is a small, cheap, hacker-style gadget made using an ESP32 microcontroller.
It’s basically a pocket-sized Wi-Fi & Bluetooth attack/analysis device.
You can think of it like a Swiss-army knife for wireless networks.

# Why I made this
As i really love cybersecurity and IoT, I like to make my own cybersecurity tools. So i am starting from basic tools, i have made an attiny-85 digispark usb as well. So the next thing is this, ESP-Marauder.
